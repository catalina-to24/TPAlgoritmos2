1 Sintaxis general de Java

Java es un lenguaje fuertemente tipado: al declarar una variable hay que fijar su tipo y éste no cambia. Ejemplo: int i = 0;, String palabra = "hola";, double precio = 9.99;.

Todo código vive dentro de una clase. Convenciones del curso:

  • Paquetes en minúsculas: package ejercicios;
  • Clases con mayúscula inicial: public class HolaMundo
  • Métodos y variables en camelCase: mifuncion, precioTotal
package ejercicios;
public class HolaMundo {
    public static void main(String[] args) {
        // comentario de una línea
        int i;
        i = 10;
    }
}

Entrada/salida estándar:

System.out.println("Ingrese un numero");
Scanner scanner = new Scanner(System.in);
int numero = scanner.nextInt();
scanner.close();

2 Funciones

Una función tiene tipo de retorno, nombre y parámetros. Si no devuelve nada, el tipo de retorno es void.

private static void saludar() {
    System.out.println("Hola");
}

private static String saludarConNombre(String nombre) {
    return "Su nombre es " + nombre;
}

Modificadores importantes:

ModificadorQué significa
publicAccesible desde cualquier clase.
privateSolo dentro de la misma clase.
staticNo necesita una instancia para ser llamado.

3 Estructuras condicionales

if / else if / else

int i = 11;
if (i > 10) {
    System.out.println(i);
} else {
    System.out.println("muy chico");
}

switch

switch (i) {
    case 1: System.out.println("Es uno"); break;
    case 2: System.out.println("Es dos"); break;
    default: break;
}
⚠ Cuidado con el break Sin break el switch cae al siguiente caso (fall-through). Suele ser una fuente típica de bugs en el parcial.

5 Vectores (arrays unidimensionales)

Un vector es un contenedor de tamaño fijo de elementos del mismo tipo. La longitud se fija al crearlo y no se puede cambiar.

// Forma 1: reserva memoria y asigna
int[] vector = new int[10];
vector[0] = 15;

// Forma 2: literal
int[] vector2 = {3, 4, 1, 2, 6};

// Recorrido
for (int i = 0; i < vector2.length; i++) {
    System.out.println(vector2[i]);
}

Se accede por índice numérico desde 0 hasta length - 1. Salir del rango lanza ArrayIndexOutOfBoundsException.

6 Acumuladores booleanos

Técnica clásica: usar una variable boolean para resumir el resultado de una iteración sobre una colección.

public static boolean tieneAlgunNumeroPar(int[] numeros) {
    boolean rta = false;
    for (int i = 0; i < numeros.length; i++) {
        if (esPar(numeros[i])) {
            rta = true;
        }
    }
    return rta;
}
💡 Variantes típicas "¿Existe al menos uno?" → arrancar en false y marcar true. "¿Lo cumplen todos?" → arrancar en true y marcar false apenas uno falle.

7 Matrices (arrays bidimensionales)

int[][] matriz = {
    {1, 2, 3},
    {4, 5, 6}
};

// Recorrido por fila
for (int i = 0; i < matriz.length; i++) {
    for (int j = 0; j < matriz[i].length; j++) {
        System.out.print(matriz[i][j] + " ");
    }
    System.out.println();
}

matriz.length es la cantidad de filas; matriz[0].length es la cantidad de columnas (asumiendo filas del mismo largo).

8 Puntos clave para el examen

  1. Diferencia entre tipo primitivo (int, double, boolean) y referencia (String, arrays, objetos).
  2. Java es fuertemente tipado: el tipo de una variable se declara y no cambia.
  3. El for clásico tiene tres partes: inicialización, condición, incremento.
  4. Los vectores tienen tamaño fijo; si necesitás crecer, usá ArrayList (tema de clases siguientes).
  5. Un método static puede llamarse sin crear una instancia de la clase.
